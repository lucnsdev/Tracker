Um projeto para um rastreador veicular sem mensalidades. Usando Esp32-H2. Modulo 4G A7670C e um modulo de GPS o LC86G da Quectel.
Os principais recursos são:
1 - Zero mensalidades. Custo apenas com hardware e operadora de dados moveis.
2 - Hospedagem gratuita no Render.com. 
3 - Sistema autonomo e inteligente que não descarrega a bateria do automovel. Avalia dados e posição para desligamento automatico.
4 - Rastreio em tempo real graças ao Firebase Cloud Messaging da Google.
